参数说明

参数 描述 参数值类型 默认值
fixedColumnCount 设置想要固定的列数 插件会自动检测设置值，当值大于表格总列数时，此值会被重置为0 数值 0
tableMaxHeight 设置表格最大高度 插件会自动检测设置值，当值大于表格总高度时，此值会被重置为表格总高度 数值 0
tableMaxWidth 设置表格最大宽度 插件会自动检测设置值，当值大于表格总宽度时，此值会被重置为表格总宽度 数值 0
fixedHeader 设置是否固定头，true : 固定（插件会自动检测thead里的行数并进行固定） 布尔 true
adjustWHSettings 设置各种高度的调整值（当表格样式复杂多样时，若出现各区域宽高不理想时候，可以指定区域微调整） 对象 -
addLeftWidth 设置固定列区域宽度添加值 数值 0
addTopHeight 设置固定头区域高度添加值 数值 0
addRightTopWidth 设置固定后右上区域宽度添加值 数值 0
addLeftBottomHeight 设置固定后左下区域宽度添加值 数值 0
submitEreas 当表格在form间，内有表单元素时，设置想要提交的区域 对象 -
leftTop 设置左上为提交区域 布尔 false
leftBottom 设置左下为提交区域 布尔 false
rightTop 设置右上为提交区域 布尔 false
rightBottom 设置右下为提交区域 布尔 true
submitSelector 设置表单提交的目标元素名（，分隔多个） 字符串 "a,input,select,textarea"

你的表格需要包含thead和tbody，然后象下边一样简单调用即可,
当然别忘记了先引入jquery库

<script>
$(document).ready(function() {
$("table").fixedheadercolumntable();
});
</script>
